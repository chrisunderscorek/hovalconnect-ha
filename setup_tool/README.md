# Hoval Connect – Setup Tool

Dieses Tool holt automatisch die Tokens für die Home Assistant Integration.

## Installation (einmalig)

1. Python installieren: https://www.python.org/downloads/
   - ✅ **"Add Python to PATH"** ankreuzen!

2. `install.bat` doppelklicken
   - Lädt Chromium (~150MB) – nur einmalig nötig

## Verwendung

1. `start.bat` doppelklicken
2. Hoval Email + Passwort eingeben
3. **"Tokens holen"** klicken
4. Access Token + Refresh Token erscheinen
5. Beide Tokens in Home Assistant einfügen

## Sicherheit

- Zugangsdaten werden **nie gespeichert**
- Browser läuft **unsichtbar** im Hintergrund
- Übertragung nur via **HTTPS** an Hoval
- Tokens werden nach Anzeige aus dem RAM gelöscht

## Anforderungen

- Windows 10/11
- Python 3.10+
- Internetverbindung
